# Mr.Moose


## Setup

```bash
$ sudo apt update
$ sudo apt install software-properties-common
$ sudo add-apt-repository ppa:deadsnakes/ppa
$ sudo apt update
$ sudo apt install python3.7 python3.7-dev python3-pip
$ pip install --user pipenv
$ pipenv install
$ pipenv shell
```


## Run

```bash
python3 puzzle.py
```
